AngularJS Circular Time Picker

Native AngularJS datetime picker element directive with a circular interface for time

Version 0.1.0

Code on GitHubDownload

Example

Inline

Invoke the datetime picker inline with your markup

How

You can pass configuration options as a function or an object.

<circulartimepicker model="datetime2"></circulartimepicker>

<circulartimepicker model="datetime2" format="{military:true}"></circulartimepicker>

<circulartimepicker model="datetime2" format="{momentFormat:'YYYY'}"></circulartimepicker>
